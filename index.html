Mathematics Tools & Resources

About These Tools

These browser-based tools are designed to support active learning in mathematics courses. They are completely client-side (no data leaves your browser), free to use, and suitable for students, instructors, and anyone exploring these mathematical topics.

Differential Equations

Interactive visualizations and numerical solvers for ordinary differential equations. Features include phase portraits, direction fields, eigenvalue analysis for linear systems, and numerical integration methods.

Explore ODE Tools

Linear Algebra

Interactive tools for matrix operations, eigenvalues/eigenvectors, linear transformations, vector spaces, and basis changes. Visual demonstrations of fundamental linear algebra concepts.

Explore Linear Algebra Tools

Numerical Methods

Interactive implementations of core numerical algorithms: root-finding (Newton, bisection), interpolation, numerical integration, least squares fitting, and finite difference methods.

Explore Numerical Tools

PDF Document Tools

Client-side utilities for managing PDF files commonly used in courses — merge multiple files, reorder or delete pages, and organize scanned homework or notes. All processing happens in your browser.

  • PDF Merger
  • PDF Organizer
Access PDF Tools

Exam Timer

A clean, distraction-free countdown timer with current time display. Useful for pacing during exams, presentations, or timed exercises. Features include customizable duration and visual time indicators.

Open Exam Timer

Grade Distributions Calculator

Compute final letter grade cutoffs based on exam scores, assignment weights, and desired distribution curves. Analyze class performance and set appropriate grading scales.

Open Grade Calculator

Seats Allocator

Generate randomized or structured seating charts for common Texas A&M mathematics classrooms. Facilitates fair exam seating arrangements and helps manage classroom organization.

Open Seats Allocator

Supplementary materials including lecture notes, practice problems, and instructional guides for mathematics courses taught at Texas A&M University.
