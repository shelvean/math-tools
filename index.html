Math Tools & Resources

These browser-based tools were developed to support learning in my courses. They run entirely in your browser (no data is sent to any server), and may be helpful for students, instructors, or anyone exploring these topics.

Differential Equations

Interactive visualizations and numerical solvers for ordinary differential equations, including phase portraits, direction fields, and eigenvalue analysis for systems.

Explore ODE Tools →

Linear Algebra

Tools for matrix operations, eigenvectors/eigenvalues, vector space visualizations, and interactive demonstrations of concepts like linear transformations and basis changes.

Explore Linear Algebra Tools →

Numerical Methods

Interactive implementations of core algorithms: root-finding (Newton, bisection), interpolation, numerical integration, least squares fitting, and finite difference schemes.

Explore Numerical Tools →

PDF Document Tools

Simple, client-side utilities for managing PDF files commonly used in courses — merge multiple files, reorder or delete pages, and organize scanned homework or notes.

  • PDF Merger
  • PDF Organizer

Exam Timer

A clean, local countdown timer with current time display — perfect for in-class exams or practice tests. No data sent anywhere; runs entirely in your browser.

Open Timer →

Grade Distributions Calculator

Compute final letter grade cutoffs based on exam scores, weights, and desired distribution curves. Useful for planning grading schemes or student self-assessment.

Open Grade Calculator →

Seats Allocator

Generate randomized or structured seating charts for common Texas A&M math classrooms (e.g., Blocker rooms) to facilitate fair exam seating.
